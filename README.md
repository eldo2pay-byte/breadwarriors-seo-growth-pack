# BreadWarriors SEO Growth Pack

This repository contains the SEO growth package prepared for BreadWarriors.

## Contents

- `breadwarriors-seo-growth-pack.zip`: Complete downloadable package.
- `hercules-ai-prompt.txt`: Plain-text instructions for Hercules AI.
- `seo-pack/`: Source files for competitor findings, alternative pages, blog drafts, programmatic SEO templates, JSON-LD and sitemap templates.

## How to Use with Hercules

Give Hercules this repository URL and ask it to use `seo-pack/10-hercules-ai-instructions.md` as the main task document.

Repository:

```txt
https://github.com/eldo2pay-byte/breadwarriors-seo-growth-pack
```

Do not deploy directly at first. Ask Hercules to prepare a patch/PR, file list, deploy commands and test plan.
